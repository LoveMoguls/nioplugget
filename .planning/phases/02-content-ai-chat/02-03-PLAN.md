**prompt.go** — Conversation builder with sliding window:
```go
const DefaultWindowSize = 20 // Keep system prompt + last 20 messages

func BuildMessageHistory(messages []queries.Message, windowSize int) []anthropic.MessageParam {
    // Apply sliding window
    if len(messages) > windowSize {
        messages = messages[len(messages)-windowSize:]
    }
    // Ensure first message is from "user" (Claude API requirement)
    // If truncation caused first message to be "assistant", drop it
    for len(messages) > 0 && messages[0].Role == "assistant" {
        messages = messages[1:]
    }
    // Convert to SDK params
    params := make([]anthropic.MessageParam, 0, len(messages))
    for _, m := range messages {
        if m.Role == "user" {
            params = append(params, anthropic.NewUserMessage(anthropic.NewTextBlock(m.Content)))
        } else {
            params = append(params, anthropic.NewAssistantMessage(anthropic.NewTextBlock(m.Content)))
        }
    }
    return params
}
```

**stream.go** — SSE streaming proxy:
```go
func StreamChatResponse(w http.ResponseWriter, r *http.Request, apiKey string, systemPrompt string, messages []anthropic.MessageParam) (string, error) {
    // Set SSE headers
    w.Header().Set("Content-Type", "text/event-stream")
    w.Header().Set("Cache-Control", "no-cache")
    w.Header().Set("Connection", "keep-alive")
    w.Header().Set("X-Accel-Buffering", "no") // nginx proxy support

    flusher, ok := w.(http.Flusher)
    if !ok {
        return "", fmt.Errorf("streaming not supported")
    }

    client := anthropic.NewClient(option.WithAPIKey(apiKey))
    stream := client.Messages.NewStreaming(r.Context(), anthropic.MessageNewParams{
        Model:     anthropic.ModelClaudeSonnet4_6,
        MaxTokens: 1024,
        System:    []anthropic.TextBlockParam{{Text: systemPrompt}},
        Messages:  messages,
    })

    var fullResponse strings.Builder
    message := anthropic.Message{}
    for stream.Next() {
        event := stream.Current()
        message.Accumulate(event)
        switch ev := event.AsAny().(type) {
        case anthropic.ContentBlockDeltaEvent:
            switch delta := ev.Delta.AsAny().(type) {
            case anthropic.TextDelta:
                fullResponse.WriteString(delta.Text)
                // Send SSE event with JSON-encoded text chunk
                data, _ := json.Marshal(map[string]string{"text": delta.Text})
                fmt.Fprintf(w, "data: %s\n\n", data)
                flusher.Flush()
            }
        }
    }
    if stream.Err() != nil {
        // Send error event
        errData, _ := json.Marshal(map[string]string{"error": "AI-tjansten svarade inte korrekt. Forsok igen."})
        fmt.Fprintf(w, "data: %s\n\n", errData)
        flusher.Flush()
        return "", stream.Err()
    }
    // Send done event
    fmt.Fprintf(w, "data: [DONE]\n\n")
    flusher.Flush()
    return fullResponse.String(), nil
}
```

**handler.go** — ChatHandler with three endpoints:

1. `POST /api/sessions` — Create a new session. Body: `{exerciseId: string}`. Requires child JWT. Creates session in DB, returns session JSON `{id, exerciseId, startedAt}`.

2. `POST /api/sessions/{id}/messages` — Send a message and stream AI response. Body: `{content: string}`. Flow:
   a. Verify session belongs to authenticated student and is not ended (ended_at IS NULL)
   b. Save user message to DB
   c. Load session's exercise to get system_prompt
   d. Look up student's parent_id, then get parent's encrypted API key
   e. Decrypt API key using EncryptionService
   f. Load all messages for session from DB
   g. Build message history with sliding window (prompt.go)
   h. Stream Claude response via SSE (stream.go)
   i. Save complete AI response to DB
   j. Return (SSE stream already sent)

3. `GET /api/sessions/{id}/messages` — List all messages for a session. Returns JSON array of messages. Verify session belongs to authenticated student.

IMPORTANT: The SSE streaming endpoint needs a longer write timeout. In main.go, either:
- Set the server WriteTimeout to 0 (disable) or a longer value like 5 minutes
- Or use `http.ResponseController` to extend the deadline for this specific handler
