<task type="auto" tdd="true">
  <name>Task 3: Register, Login, Logout HTTP handlers</name>
  <files>backend/internal/auth/handler.go, backend/internal/auth/handler_test.go</files>
  <behavior>
    - POST /api/auth/register with valid {email, password, gdprConsent: true} returns 201 + sets JWT cookie
    - POST /api/auth/register with existing email returns 409
    - POST /api/auth/register with password < 8 chars returns 400
    - POST /api/auth/register without gdprConsent returns 400
    - POST /api/auth/login with valid credentials returns 200 + sets JWT cookie
    - POST /api/auth/login with wrong password returns 401
    - POST /api/auth/login with unknown email returns 401 (same message as wrong password)
    - POST /api/auth/logout clears JWT cookie, returns 200
    - JWT cookie is httpOnly, Secure, SameSite=Lax, Path=/
  </behavior>
  <action>
    1. Create `internal/auth/handler.go`:
       - `type AuthHandler struct { queries *queries.Queries; tokenAuth *jwtauth.JWTAuth }`
       - `NewAuthHandler(q *queries.Queries, ta *jwtauth.JWTAuth) *AuthHandler`

       - `Register(w, r)`:
         - Decode JSON: {email, password, gdprConsent}
         - Validate: email not empty, password >= 8 chars, gdprConsent must be true
         - Hash password with HashPassword
         - Insert via queries.CreateParent(email, hash, gdpr_consent_at=now)
         - On duplicate email: return 409 {"error": "E-postadressen ar redan registrerad"}
         - Generate JWT with role "parent"
         - Set httpOnly cookie: name="token", value=jwt, httpOnly=true, secure=true, sameSite=Lax, path="/", maxAge=86400
         - Return 201 {"id": parentID, "email": email}

       - `Login(w, r)`:
         - Decode JSON: {email, password}
         - Look up parent by email
         - Not found or wrong password: return 401 {"error": "Fel e-post eller losenord"} (same message for both — prevent user enumeration)
         - Generate JWT, set cookie (same as register)
         - Return 200 {"id": parentID, "email": email}

       - `Logout(w, r)`:
         - Clear cookie: set same cookie name with maxAge=-1
         - Return 200 {"message": "Utloggad"}

    2. Wire handlers into main.go router:
       - Public: POST /api/auth/register, POST /api/auth/login
       - Protected (parent): POST /api/auth/logout

    3. Write handler tests using httptest.NewServer with a test database or mock queries.
       Use an interface for the queries to enable test mocking.
  </action>
  <verify>
    <automated>cd backend && go test ./internal/auth/ -run TestHandler -v</automated>
  </verify>
  <done>All three auth endpoints work: register creates account + JWT cookie, login validates + JWT cookie, logout clears cookie. GDPR consent timestamp stored.</done>
