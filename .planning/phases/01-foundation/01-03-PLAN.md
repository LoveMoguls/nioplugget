    - NewEncryptionService with key != 32 bytes returns error

    - Nonce: crypto/rand, aesGCM.NonceSize() bytes
    - Seal: prepend nonce to ciphertext
    - Open: split nonce from ciphertext

    - ValidateAPIKey with valid key format makes HTTP GET to https://api.anthropic.com/v1/models and returns nil on 200
    - ValidateAPIKey with invalid key returns descriptive error
    - ValidateAPIKey with empty string returns error without making HTTP call
    - MaskAPIKey("sk-ant-api03-abcdef...xyz") returns "sk-ant-...****"
    - MaskAPIKey with short key returns "****"
  </behavior>
  <action>
    Create `internal/apikey/validate.go`:
    - `ValidateAPIKey(apiKey string) error`:
      - Reject empty key
      - HTTP GET to https://api.anthropic.com/v1/models
      - Headers: x-api-key: {apiKey}, anthropic-version: 2023-06-01
      - 200: valid, return nil

      - Connection error: return "Kunde inte ansluta till Anthropic"
      - Use a 10-second timeout on the HTTP client

    - `MaskAPIKey(key string) string`:
      - If len < 8: return "****"
      - Return first 6 chars + "...****"
      - Example: "sk-ant-api03-abc..." -> "sk-ant...****"

<task type="auto" tdd="true">
  <name>Task 3: API key CRUD handlers</name>
  <files>backend/internal/apikey/handler.go, backend/internal/apikey/handler_test.go</files>
  <behavior>
    - POST /api/apikey with valid key: validates key, encrypts, stores, returns 201 with masked key
    - POST /api/apikey with invalid key: returns 400 with validation error message
    - GET /api/apikey: returns 200 with masked key and timestamps (never returns raw key)
    - GET /api/apikey when no key stored: returns 404
    - PUT /api/apikey with valid key: validates, re-encrypts, updates, returns 200 with masked key
    - DELETE /api/apikey: deletes key, returns 200
    - All endpoints require parent JWT (tested via middleware)
  </behavior>
  <action>
    Create `internal/apikey/handler.go`:
    - `type APIKeyHandler struct { queries *queries.Queries; encSvc *EncryptionService }`
    - `NewAPIKeyHandler(q, enc) *APIKeyHandler`

    - `Store(w, r)` — POST:
      - Extract parent_id from JWT context (auth.GetUserIDFromContext)
      - Decode JSON: {apiKey: string}
      - Validate key via ValidateAPIKey (live Claude API call)
      - On validation failure: return 400 {"error": "API-nyckeln ar inte giltig. Kontrollera att du kopierat ratt nyckel fran console.anthropic.com"}
      - Encrypt key via encSvc.Encrypt
      - Store via queries.UpsertAPIKey(parent_id, encrypted_key)
      - Return 201 {"maskedKey": MaskAPIKey(apiKey), "message": "API-nyckel sparad"}

    - `Get(w, r)` — GET:
      - Get encrypted key from DB
      - Decrypt to get raw key (only for masking — NEVER return raw)
      - Return 200 {"maskedKey": MaskAPIKey(rawKey), "createdAt": ..., "updatedAt": ...}

    - `Update(w, r)` — PUT:
      - Same as Store (UpsertAPIKey handles update)
      - Return 200 instead of 201

    - `Delete(w, r)` — DELETE:
      - queries.DeleteAPIKeyByParentID
      - Return 200 {"message": "API-nyckel borttagen"}

    Wire into main.go:
    - Protected route group (parent JWT required):
      POST /api/apikey -> Store
      GET /api/apikey -> Get
      PUT /api/apikey -> Update
      DELETE /api/apikey -> Delete
  </action>

  </verify>
