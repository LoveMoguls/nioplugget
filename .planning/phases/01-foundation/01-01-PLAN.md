<task type="auto">
  <name>Task 2: Create database schema migration and sqlc configuration</name>
  <files>
    backend/db/migrations/001_initial_schema.up.sql,
    backend/db/migrations/001_initial_schema.down.sql,
    backend/db/queries/parents.sql,
    backend/db/queries/students.sql,
    backend/db/queries/api_keys.sql,
    backend/sqlc.yaml
  </files>
  <action>
    1. Create `db/migrations/001_initial_schema.up.sql`:
       ```sql
       CREATE EXTENSION IF NOT EXISTS "pgcrypto";

       CREATE TABLE parents (
           id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
           email TEXT NOT NULL UNIQUE,
           password_hash TEXT NOT NULL,
           gdpr_consent_at TIMESTAMPTZ NOT NULL,
           created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
       );

       CREATE TABLE students (
           id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
           parent_id UUID NOT NULL REFERENCES parents(id) ON DELETE CASCADE,
           name TEXT NOT NULL,
           pin_hash TEXT,
           invite_code TEXT UNIQUE,
           invite_expires_at TIMESTAMPTZ,
           activated_at TIMESTAMPTZ,
           created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
       );

       CREATE TABLE api_keys (
           id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
           parent_id UUID NOT NULL UNIQUE REFERENCES parents(id) ON DELETE CASCADE,
           encrypted_key BYTEA NOT NULL,
           created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
           updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
       );

       CREATE TABLE pin_attempts (
           student_id UUID PRIMARY KEY REFERENCES students(id) ON DELETE CASCADE,
           attempts INT NOT NULL DEFAULT 0,
           locked_until TIMESTAMPTZ,
           last_attempt_at TIMESTAMPTZ
       );
       ```
    2. Create `db/migrations/001_initial_schema.down.sql`:
       ```sql
       DROP TABLE IF EXISTS pin_attempts;
       DROP TABLE IF EXISTS api_keys;
       DROP TABLE IF EXISTS students;
       DROP TABLE IF EXISTS parents;
       ```
    3. Create `db/queries/parents.sql` with sqlc annotations:
       - CreateParent (insert email, password_hash, gdpr_consent_at)
       - GetParentByEmail (select by email)
       - GetParentByID (select by id)
    4. Create `db/queries/students.sql` with sqlc annotations:
       - CreateStudent (insert parent_id, name, invite_code, invite_expires_at)
       - GetStudentsByParentID (list children)
       - ActivateStudent (atomic UPDATE: set pin_hash, activated_at, clear invite_code WHERE invite_code = $1 AND activated_at IS NULL AND invite_expires_at > NOW())
       - GetStudentByNameAndParent (for login name selection)
       - GetStudentByID
       - ListStudentNamesByParentID (for child login name list)
    5. Create `db/queries/api_keys.sql` with sqlc annotations:
       - UpsertAPIKey (insert or update encrypted_key for parent_id)
       - GetAPIKeyByParentID (select encrypted_key)
       - DeleteAPIKeyByParentID
    6. Create `sqlc.yaml`:
       ```yaml
       version: "2"
       sql:
         - engine: "postgresql"
           queries: "db/queries/"
           schema: "db/migrations/"
           gen:
             go:
               package: "queries"
               out: "internal/database/queries"
               sql_package: "pgx/v5"
               emit_json_tags: true
               emit_prepared_queries: false
               emit_interface: false
               emit_exact_table_names: false
       ```
    7. Run `sqlc generate` to produce Go code in internal/database/queries/
  </action>
  <verify>
    <automated>cd backend && go install github.com/sqlc-dev/sqlc/cmd/sqlc@latest && sqlc generate && go build ./...</automated>
  </verify>
  <done>Migration files exist, sqlc generates type-safe Go query code, project compiles with generated code</done>
</task>

    1. Create `internal/middleware/logger.go` using rs/zerolog:

       - Log at Info level for 2xx/3xx, Warn for 4xx, Error for 5xx
