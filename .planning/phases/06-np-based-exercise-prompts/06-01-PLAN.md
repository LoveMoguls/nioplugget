**NP-calibrated prompt structure for Samhällskunskap:**

Same base structure as Biologi but adapted for SO:
- Reference Delprov A and B (both mix faktafrågor + resonerande frågor with source material)
- The word "resonera" is THE key NP verb for samhälle — use it prominently in C/A-level prompts
- Questions should specify quantity like real NP: "Ge 1-2 fördelar", "Resonera om 1-3 orsaker"
- Build the "resonera"-skill explicitly: guide students through position → reason → example → counterpoint

**Subject-specific NP patterns from research:**
- E-level: "Vad menas med...?", "Para ihop...", matching/definition tasks
- C-level: "Resonera om fördelar och nackdelar med...", "Beskriv orsaker till att...", "Varför...?"
- A-level: "Resonera kring varför det ser ut som... visar", "Ta ställning... Motivera.", "Välj ut [N] och resonera om..."
- Use real Swedish civic contexts: folkomröstningen, riksdagen, kommunal vs statlig verksamhet
- Use suggested NP terms in questions: "kön, etnicitet, ekonomi, social bakgrund"
- Source-based question pattern: "Tabellen visar... Resonera kring varför..."

**Common student mistakes to address (from NP research):**
- Writing too little on resonerande frågor (E-level answer when C/A needed)
- Not using subject-specific vocabulary ("Folk är fattiga" vs "ekonomisk ojämlikhet")
- Confusing EU, FN, and Nordiska rådet
- One-sided answers when asked for fördelar och nackdelar
- Mixing up juridical concepts (brottmål vs civilmål)
- Source analysis weakness (stating what table shows without explaining why)
- Ignoring the specified quantity ("ge 1-2 orsaker")

**Scoring awareness for C/A prompts:**
- E-svar: Nämner en relevant orsak utan förklaring
- C-svar: Förklarar med orsak + belägg, använder ämnesspecifikt språk korrekt
- A-svar: Flera perspektiv, nyanserat argument, väger fördelar mot nackdelar, använder precisa samhällsbegrepp

**NP-calibrated prompt structure for Matematik:**

Math NP prompts differ fundamentally from bio/samhälle:
- Reference specific delprov: B (utan hjälpmedel), C (med hjälpmedel), D (fullständiga lösningar)
- Delprov D is where grade differentiation happens — prompts at C/A-level should mirror its style
- PRIM-gruppens bedömningsprincip: "Utgångspunkten för bedömningen är att eleven ska få poäng för lösningens förtjänster, inte avdrag för fel"
- Always ask "Visa hur du tänker" / "Förklara hur du kom fram till svaret"
- Use concrete contexts with real names: "Lena", "Ali", "Maria" doing real things
- Word problems in Delprov D style for C/A exercises

**Subject-specific NP patterns from research:**
- E-level: "Beräkna...", "Lös ekvationen...", "Bestäm arean av...", "Vilket påstående stämmer?"
- C-level: "Visa hur du beräknar...", "Förklara varför...", "Skriv en ekvation som beskriver sambandet..."
- A-level: "Förklara vad lutningen i grafen betyder i det här sammanhanget", "Är [påstående] rimligt? Motivera.", real-world word problems requiring modeling

**Common student mistakes to address (from NP research):**
- Percentage confusion: 25% decrease then 25% increase ≠ original (different base values)
- Not showing work in Delprov D (right answer but zero method = fewer points)
- Linear function slope: calculating correctly but not explaining meaning in context
- Confusing medelvärde, median, typvärde under pressure
- Pythagoras: misidentifying the hypotenuse in applied problems
- Unit conversion errors (cm to m, liter to deciliter)
- Word-to-algebra translation failure
- Proportionality vs linearity confusion
