<decisions_made>

- Gamification approach B chosen: dedicated challenge track, stars (1-3) + XP per exercise, sequential unlocking
- Target: 13-year-old girl with ADD — short bursts, immediate visual reward
- 5-6 images max (user changed from 1-3 during brainstorming)
- Claude auto-generates everything (title, emoji, 4-6 exercises, system prompts)
- Separate track from regular subjects (not integrated into existing subjects/topics)
- LATERAL join fallback: ListChallengeExercisesWithProgress uses regular LEFT JOIN (not LATERAL) — sqlc handled it fine
- XP formula: 1-2 score → 1 star/10 XP, 3-4 score → 2 stars/20 XP, 5 score → 3 stars/30 XP
- Back navigation: chat page uses ?returnTo=/challenges/[id] URL param, redirects with ?stars=N&xp=N after session ends
- SM-2 spaced repetition skipped for challenge sessions
</decisions_made>
